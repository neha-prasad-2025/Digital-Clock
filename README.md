# Digital Clock ⏰

A modern digital clock built using HTML, CSS, and JavaScript.

## Features

- Live time updates
- 12-hour format with AM/PM support
- Leading zeros for single-digit values
- Responsive design for mobile and desktop devices
- Dark mode toggle
- Theme persistence using Local Storage

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## What I Learned

- DOM manipulation
- Working with the Date object
- setInterval()
- Event handling
- CSS media queries
- Local Storage
- Debugging and problem-solving
